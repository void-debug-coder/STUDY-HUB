14. Stress & Strain in Bars of Varying Length

Rule: Total elongation = Sum of elongation of each part
ΔL_total = Σ(P×L_i)/(A_i×E_i) = σ₁L₁/E₁ + σ₂L₂/E₂ + ...

If same P: ΔL = (P/E)(L₁/A₁ + L₂/A₂ + ...)

16. Thermal Stress - Basic Cases

Case 1: Free Expansion
ΔL_free = L×α×ΔT, ε = α×ΔT, σ = 0

Case 2: Fully Restrained
σ = E×α×ΔT, F = A×E×α×ΔT
Heating → Compression, Cooling → Tension

Case 3: Supports Yield by Δ
σ = E(α×ΔT - Δ/L)

α Values: Steel: 12×10⁻⁶, Brass: 19×10⁻⁶, Al: 24×10⁻⁶, Concrete: 10×10⁻⁶ /°C

24. Stepped Bars Fixed Ends

Strain Compatibility: α×ΔT×ΣL = P/E × Σ(L_i/A_i)
Force: P = E×α×ΔT×ΣL / Σ(L_i/A_i)
Stresses: σ_i = P/A_i
Key: Smaller A → Higher σ

27. Composite Bars - Different Materials

Conditions:
1. Strain: α₁ΔT + σ₁/E₁ = α₂ΔT + σ₂/E₂
2. Force: A₁σ₁ + A₂σ₂ = 0 (or = P if external load)

Formula (No load):
σ₁ = E₁E₂A₂(α₂-α₁)ΔT / (A₁E₂+A₂E₁)
σ₂ = -A₁σ₁/A₂

Rule: Higher α material → Compression

32. Rod & Tube Composite

Setup: Rod inside tube, screwed/nutted at ends. Heated ΔT.

Rule: Material with HIGHER α → Compression
Material with LOWER α → Tension

Equations:
1. Force balance: A_rod×σ_rod + A_tube×σ_tube = 0
2. Strain: α_rod×ΔT + σ_rod/E_rod = α_tube×ΔT + σ_tube/E_tube

33. Thermal + External Load Combined

When composite carries load P + heated ΔT:
A₁σ₁ + A₂σ₂ = P
α₁ΔT + σ₁/E₁ = α₂ΔT + σ₂/E₂

Solution:
σ₁ = [P×E₁ + E₁E₂A₂(α₂-α₁)ΔT] / (A₁E₂+A₂E₁)
σ₂ = [P×E₂ + E₁E₂A₁(α₁-α₂)ΔT] / (A₁E₂+A₂E₁)

Superposition: Total stress = Load stress + Thermal stress

Complete Worked Examples

Example 1: Restrained Steel Bar
Steel bar 2m long, 50mm dia, rigidly held. Temp increases 50°C. E=200GPa, α=12×10⁻⁶/°C

Solution:
σ = E×α×ΔT = 200,000×12×10⁻⁶×50 = 120 N/mm² (Compression)
F = A×σ = π×50²/4 × 120 = 235.6 kN
Example 2: Stepped Bar
A₁=400mm², L₁=500mm, A₂=600mm², L₂=200mm. E=200GPa, α=12×10⁻⁶, ΔT=15°C

Solution:
P = 200,000×12×10⁻⁶×15×700/(500/400+200/600) = 15,916 N
σ₁ = 15,916/400 = 39.8 N/mm² (Compression)
σ₂ = 15,916/600 = 26.5 N/mm² (Compression)
Example 3: Gun Metal Rod in Steel Tube
Gun metal rod 20mm dia in steel tube 30mm OD, 25mm ID. ΔT=200°F.
α_gm=10×10⁻⁶/°F, E_gm=100GPa, α_s=6×10⁻⁶/°F, E_s=200GPa

Solution:
A_gm = 314.16mm², A_s = 215.98mm²
σ_gm = 100,000×200,000×215.98×(6-10)×10⁻⁶×200/(314.16×200,000+215.98×100,000)
σ_gm = -46.3 N/mm² (Compression)
σ_s = +67.4 N/mm² (Tension)
Check: 314.16×(-46.3) + 215.98×67.4 ≈ 0 ✓
Exam Tips:
1. Always check units: E in N/mm² or GPa, α in /°C or /°F
2. Heating → Compression if restrained, Cooling → Tension
3. Higher α material → Compression in composites
4. For stepped bars, force P is constant, stress varies with 1/A
5. Always verify: Σ(A×σ) = External Load (or 0)
