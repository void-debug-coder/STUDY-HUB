14. Stress & Strain in Bars of Varying Length

Rule: Total elongation = Sum of elongation of each part
ΔL_total = Σ(P×L_i)/(A_i×E_i) = σ₁L₁/E₁ + σ₂L₂/E₂ + ...

If same P: ΔL = (P/E)(L₁/A₁ + L₂/A₂ + ...)
Example 23: Stepped Bar
A bar has section AB: L₁ = 500mm, A₁ = 1000mm² and BC: L₂ = 800mm, A₂ = 500mm².
Carries axial load P = 50 kN. E = 200 GPa. Find total elongation.

Solution:
σ₁ = P/A₁ = 50,000/1000 = 50 N/mm²
σ₂ = P/A₂ = 50,000/500 = 100 N/mm²

ΔL = σ₁L₁/E + σ₂L₂/E
ΔL = (50×500)/200,000 + (100×800)/200,000
ΔL = 0.125 + 0.4 = 0.525 mm

16. Thermal Stress - Basic Cases

Case 1: Free Expansion
ΔL_free = L×α×ΔT, ε = α×ΔT, σ = 0

Case 2: Fully Restrained
σ = E×α×ΔT, F = A×E×α×ΔT
Heating → Compression, Cooling → Tension

Case 3: Supports Yield by Δ
σ = E(α×ΔT - Δ/L)

α Values: Steel: 12×10⁻⁶, Brass: 19×10⁻⁶, Al: 24×10⁻⁶, Concrete: 10×10⁻⁶ /°C
Example 20: Aluminum Bar Fixed Ends
An aluminum bar fixed at both ends is heated through 20K. Find stress developed.
Take E = 80 GPa = 80,000 N/mm², α = 24×10⁻⁶ /K

Solution:
Since fixed both ends, expansion prevented → Compressive stress
σ = E×α×ΔT
σ = 80,000 × 24×10⁻⁶ × 20
σ = 80,000 × 0.00048
σ = 38.4 N/mm² (Compressive)
Example 22: Max Temperature Rise - Brass Rod
A brass rod 20m long is fixed at both ends. If thermal stress is not to exceed 76.5 N/mm², calculate the temperature through which the bar should be heated. Take α = 17×10⁻⁶ /°C, E = 90 kN/mm²

Given:
L = 20m = 20,000mm, σ_max = 76.5 N/mm²
α = 17×10⁻⁶ /°C, E = 90,000 N/mm²

Solution:
For fixed bar: σ = E×α×ΔT
76.5 = 90,000 × 17×10⁻⁶ × ΔT
76.5 = 1.53 × ΔT
ΔT = 50°C

Answer: Bar can be heated by 50°C maximum

24. Stepped Bars Fixed Ends

Rule: For stepped bar fixed at both ends, thermal force P is constant throughout.

Strain Compatibility: Total free expansion = Total contraction by force P
α×ΔT×(L₁+L₂+...) = P/E × (L₁/A₁ + L₂/A₂ + ...)

Solve for P: P = E×α×ΔT×(L₁+L₂) / (L₁/A₁ + L₂/A₂ + ...)

Then stresses: σ₁ = P/A₁, σ₂ = P/A₂, etc.
Key: Smaller area → Higher thermal stress
Example 24: Stepped Steel Rod
Steel rod AB firmly held between rigid supports. A₁=400mm², L₁=500mm, A₂=600mm², L₂=200mm. Heated 15K. Find stress. α=12×10⁻⁶/°C, E=200 GPa

Solution:
Step 1: Free expansion
ΔL_free = α×ΔT×(L₁+L₂) = 12×10⁻⁶×15×700 = 0.126mm

Step 2: Force P to prevent expansion
0.126 = P/200,000 × (500/400 + 200/600)
0.126 = P/200,000 × 1.5833
P = 15,916 N

Step 3: Stresses
σ₁ = 15,916/400 = 39.8 N/mm² (Compressive)
σ₂ = 15,916/600 = 26.5 N/mm² (Compressive)

27. Thermal Stress in Composite Bars - Different Materials

Concept: Different materials have different α. When heated together, they want to expand different amounts but are forced to expand together.

Result: Material with higher α goes into COMPRESSION. Material with lower α goes into TENSION.

Conditions:
1. Strain Compatibility: Final strain same for both = α₁ΔT + σ₁/E₁ = α₂ΔT + σ₂/E₂
2. Force Equilibrium: A₁σ₁ + A₂σ₂ = 0 (no external load)

Solving: σ₁ = E₁E₂A₂(α₂-α₁)ΔT/(A₁E₂+A₂E₁)
Example 25: Steel Core in Brass Tube
Steel rod A_s=500mm² inside brass tube A_b=1000mm², rigidly connected.
Heated 60°C. Find thermal stresses.

Given:
Steel: E_s=200GPa, α_s=12×10⁻⁶/°C
Brass: E_b=100GPa, α_b=19×10⁻⁶/°C
ΔT = 60°C

Solution:
α_b > α_s, so brass wants to expand more → brass in compression, steel in tension

σ_s = E_s×E_b×A_b(α_b-α_s)ΔT / (A_s×E_b + A_b×E_s)
σ_s = 200,000×100,000×1000×(19-12)×10⁻⁶×60 / (500×100,000 + 1000×200,000)
σ_s = 8.4×10¹² / 250×10⁶ = 33.6 N/mm² (Tension)

σ_b = -A_s×σ_s/A_b = -500×33.6/1000 = -16.8 N/mm² (Compression)

Check: 500×33.6 + 1000×(-16.8) = 16,800 - 16,800 = 0 ✓

32. Rod & Tube Composite

Setup: Rod inside tube, screwed/nutted at ends. Heated ΔT.

Rule: Material with HIGHER α → Compression
Material with LOWER α → Tension

Equations:
1. Force balance: A_rod×σ_rod + A_tube×σ_tube = 0
2. Strain: α_rod×ΔT + σ_rod/E_rod = α_tube×ΔT + σ_tube/E_tube
Example 29: Gun Metal Rod in Steel Tube
Gun metal rod 20mm dia passes through steel tube 30mm OD, 25mm ID.
Nuts screwed tightly. Temp rises 200°F. Find stresses.

Given:
A_gm = π×20²/4 = 314.16 mm², α_gm=10×10⁻⁶/°F, E_gm=100,000 N/mm²
A_s = π/4×(30²-25²) = 215.98 mm², α_s=6×10⁻⁶/°F, E_s=200,000 N/mm²
ΔT = 200°F

Solution:
1. Equilibrium: 314.16×σ_gm + 215.98×σ_s = 0
∴ σ_s = -1.455×σ_gm

2. Compatibility:
10×10⁻⁶×200 + σ_gm/100,000 = 6×10⁻⁶×200 + σ_s/200,000
0.002 + σ_gm/100,000 = 0.0012 + σ_s/200,000

3. Solve: Sub σ_s = -1.455×σ_gm
σ_gm[1/100,000 + 1.455/200,000] = -0.0008
σ_gm = -46.3 N/mm² (Compression)
σ_s = +67.4 N/mm² (Tension)

Check: 314.16×(-46.3) + 215.98×67.4 ≈ 0 ✓

33. Thermal + External Load Combined

When composite carries load P + heated ΔT:
A₁σ₁ + A₂σ₂ = P
α₁ΔT + σ₁/E₁ = α₂ΔT + σ₂/E₂

Solution:
σ₁ = [P×E₁ + E₁E₂A₂(α₂-α₁)ΔT] / (A₁E₂+A₂E₁)
σ₂ = [P×E₂ + E₁E₂A₁(α₁-α₂)ΔT] / (A₁E₂+A₂E₁)

Superposition: Total stress = Load stress + Thermal stress

Exam Tips:
1. Always check units: E in N/mm² or GPa, α in /°C or /°F
2. Heating → Compression if restrained, Cooling → Tension
3. Higher α material → Compression in composites
4. For stepped bars, force P is constant, stress varies with 1/A
5. Always verify: Σ(A×σ) = External Load (or 0)
